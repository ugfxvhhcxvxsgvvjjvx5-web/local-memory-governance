# LOW_RISK_REHOME_RULES

> 定稿时间：2026-03-28
> 目的：明确"什么才算低风险拟归位对象"，避免扩大解释
> 核心原则：**进入清单 ≠ 立即移动。清单只是执行前候选，后续需人工确认或单独批准批次。**

---

## 可进入低风险拟归位清单的

| 类型 | 典型扩展名 | 判定条件 |
|------|-----------|----------|
| 已冻结研究文档 | .md | 文件名明确标注研究主题，不含可执行逻辑 |
| 已冻结报告文档 | .md | 报告/总结/分析类，纯文本内容 |
| 普通文本笔记 | .txt | 不是脚本、不含可执行指令 |
| 普通截图 / 图片 | .png / .jpg / .jpeg / .gif / .bmp / .webp / .svg | 不是应用图标或程序资源 |
| 明确的临时 HTML 抓取 | .html / .htm | 文件名含 temp- 或明确是页面缓存，不是应用界面 |
| 明确的研究 JSON 原始结果 | .json | 内容是扫描/采集数据，不是工具配置（体量通常 >10KB） |
| 普通文档 | .docx / .xlsx / .pptx / .pdf | 不含宏、不是模板 |

### 额外判定条件

进入清单的对象还必须满足：
* 不被任何脚本引用
* 不被任何快捷方式指向
* 不含 dll / exe / 可执行伴随文件
* 不是工具的配置文件
* 不在 HIGH_RISK_ASSET_REGISTRY.csv 中

## 不能进入低风险拟归位清单的

| 类型 | 原因 |
|------|------|
| 所有高风险运行资产 | 受 HIGH_RISK_NO_MOVE_RULES.md 约束 |
| .exe / .msi / .bat / .cmd / .ps1 / .vbs / .ahk / .sh | 可执行文件一律不入此清单 |
| .dll / .ini | 运行依赖文件 |
| .json 配置文件 | 判定标准：体量小(<5KB)、名含 config/settings/rc、被工具读取 |
| 解压后的软件目录 | 整体运行依赖 |
| 游戏目录 / 修改器 / trainer / mod | 路径绑定风险 |
| 工具脚本 | 可能被计划任务或其他脚本调用 |
| 数据库文件 (.db / .sqlite) | 可能被应用读写 |
| Obsidian Vault | 已判决固定路径 |
| memory system 目录 | 已判决固定路径 |
| zip / rar / 安装包 | 需单独确认是否已安装/解压 |
| 不确定用途的对象 | 先标候审，不要进入低风险清单 |

## 如何判断 JSON 是"研究数据"还是"配置文件"

| 特征 | 研究数据 | 配置文件 |
|------|---------|---------|
| 体量 | 通常 >10 KB | 通常 <5 KB |
| 文件名 | 含 scan / result / data / raw | 含 config / settings / rc |
| 内容 | 数组/列表为主 | 键值对为主 |
| 被工具读取 | 否 | 是 |
| 可以单独打开 | 是 | 移走后工具报错 |

拿不准的一律标 `needs_manual_confirm = yes`。

---

## 执行约束

* 进入清单 ≠ 立即移动
* 清单只是执行前候选
* 后续需人工确认或单独批准批次
* 高风险资产绝不进入此清单
* 每轮执行时必须按 `LOW_RISK_MOVE_MANIFEST.csv` 逐条执行，跳过 `needs_manual_confirm = yes` 除非已获批准

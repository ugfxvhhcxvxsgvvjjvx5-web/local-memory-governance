# 给 Opus4.6 的扫描任务

你现在先不要整理磁盘，不要删除，不要搬家。

你先做的是：

扫盘 + 盘点 + 汇报

这轮重点扫描下面这些东西：

A. 工作区

* 代码仓库目录
* 反复迁移后的旧工作区
* D 盘上互相重叠的工作区
* 明显像当前主工作区的目录

B. 关键资产

* .git 仓库
* .db / .sqlite / .sqlite3
* .json 导出
* .md / .txt 文档
* 脚本文件（.py .ps1 .bat .sh）
* Obsidian 库或笔记目录
* 下载目录里的大文件
* 桌面和 HOME 根目录的大量散落文件

C. 混乱来源

* 重复下载
* 重复安装包
* 旧日志
* 临时脚本
* 根目录散落文件
* 工作区地址反复切换留下的残骸

这轮必须输出 4 个文件：

1. 02_reports\DISK_SCAN_RAW.md
2. 02_reports\WORKSPACE_MAP.md
3. 03_registry\PATH_REGISTRY.csv
4. 04_handoff\FOR_CHATGPT.md

这轮禁止做的事：

* 不删除
* 不大规模移动
* 不替用户拍板最终主工作区
* 不开始做大而全的仪表盘

扫描完成后必须回答：

1. 当前最像主工作区的是哪一个？
2. 哪几个工作区其实互相重叠？
3. 最乱的三个地方在哪里？
4. 哪些东西明显属于"只收不判"留下的堆积？
5. 下一步最应该先做资产清单还是先做安全清理？
